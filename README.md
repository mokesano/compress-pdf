# PDF Compress

PDF Compress adalah aplikasi web gratis dan sumber terbuka yang memungkinkan Anda mengompres file PDF secara online dengan cepat dan aman. Aplikasi ini dirancang untuk membantu Anda mengurangi ukuran file PDF agar lebih mudah dibagikan melalui email, diunggah ke situs web, atau disimpan di perangkat Anda tanpa mengorbankan kualitas dokumen.
Fitur Utama

    🔧 Kompresi PDF otomatis dan efisien

    🛡️ Proses kompresi dilakukan secara lokal di browser Anda, menjaga privasi dokumen

    💻 Tidak memerlukan instalasi perangkat lunak tambahan

    🌐 Akses langsung melalui browser di berbagai platform (Windows, macOS, Linux, Android, iOS)

    📁 Dukungan untuk file PDF dengan berbagai ukuran dan konten

Cara Menggunakan

    Kunjungi https://apps.sangia.org/tools/pdf/pdf-compress

    Unggah file PDF yang ingin Anda kompres

    Tunggu hingga proses kompresi selesai

    Unduh file PDF yang telah dikompres

Teknologi yang Digunakan

    HTML5, CSS3, dan JavaScript untuk antarmuka pengguna

    WebAssembly (WASM) untuk pemrosesan PDF di sisi klien

    pdf-lib untuk manipulasi file PDF
